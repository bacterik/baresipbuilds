# Baresip Builds

Files/packages will install under /opt (bin, lib, include, share, etc).

Built from source: https://github.com/baresip

Package names under the Linux APT/DNF/RPM managers are called:
"baresip-z10" & "libre-z10" for somewhat unique names. Packges are built for Debian & Fedora systems.

Under macOS it's called "z10.pkg.baresip" "z10.pkg.baresip.libre"
